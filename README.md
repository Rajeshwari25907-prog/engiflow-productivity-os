# EngiFlow OS
A specialized productivity dashboard for engineering students.

## Features
- **Weekly Sector Matrix**: Track Core Lectures, Labs, and CAD work.
- **Dynamic Analytics**: Real-time yield calculation based on grid input.
- **Performance Visualization**: Integrated Chart.js for intensity tracking.

## Tech Stack
- HTML5 / Tailwind CSS
- JavaScript (ES6)
- Chart.js / Lucide Icons
